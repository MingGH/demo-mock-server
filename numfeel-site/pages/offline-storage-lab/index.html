第二幕 / Service Worker 三种策略：同样的资源，不同取法，断网时表现天差地别。先点「加载资源」看首次，再点「模拟断网后再加载」看差异。

缓存策略对比实验

Cache First
图片 + 文字
—
先翻冰箱：有剩菜直接吃，不叫外卖。适合不常变的静态资源。
Network First
图片 + 文字
—
先叫外卖：优先等新鲜的，超时了再看冰箱。适合 API 数据、动态内容。
Stale While Revalidate
图片 + 文字
—
先吃剩菜，同时叫外卖：立刻给旧内容，后台静默更新。适合内容频率中等的页面。
没有万能策略——静态资源用 Cache First，API 数据用 Network First，内容页用 SWR。
