当前网络状态： 检测中…
点下方按钮加载资源
原理：Service Worker 在你首次打开时就把关键资源存入了 Cache Storage。之后每次请求，SW 先查缓存——有就直接返回，根本不走网络。所以断网了也能加载。

三种缓存策略速览

Cache First

先翻冰箱，有就吃，没有才叫外卖。本页用的就是这个。

Network First

先叫外卖，超时了再看冰箱。适合需要最新数据的 API。

Stale While Revalidate

先吃剩菜保证速度，同时叫外卖更新。下次就是新的。
