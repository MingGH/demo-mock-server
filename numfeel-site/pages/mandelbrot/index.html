Julia 集联动

在上方画布上移动鼠标，下方实时显示对应参数 c 的 Julia 集。曼德勃罗集是所有 Julia 集的"目录"：集合内部的 c 对应连通的 Julia 集，外部的 c 对应断裂的 Julia 集。

当前参数：

c = -0.5 + 0i

Julia 集公式：z → z² + c，其中 c 固定，z₀ 遍历整个复平面。和曼德勃罗集的区别仅在于「谁是变量，谁是参数」。

为什么两个符号能生成无限复杂

01

迭代放大差异

z² + c 每次执行都是「取平方再偏移」。两个初始值差 0.0001，经过 100 次迭代后命运完全不同——一个逃逸到无穷，一个永远困在原地。

02

边界无限复杂

集合的边界（黑色与彩色的交界处）具有无限细节。无论你放大多少倍，边界永远不会变成一条光滑曲线——这就是分形维度。

03

自相似但不完全相同

放大后会看到"小曼德勃罗集"反复出现，但每一个周围的装饰图案都略有不同。数学家证明了完全自相似的分形反而很简单（如 Sierpinski 三角），正是这种"几乎相似"制造了无穷的新意。

关键洞察

曼德勃罗集的面积是有限的（约 1.5065...），周长却是无穷大。一个有限面积的图形拥有无穷长的边界——这在欧几里得几何中不可能，但分形几何中却是自然的。你在上方每次放大，都是在探索这条无穷长的边界线。

数学定义

给定复数 c = a + bi，定义迭代序列：

z₀ = 0，zn+1 = zn² + c

曼德勃罗集 M = { c ∈ ℂ : limn→∞ |zn| ≠ ∞ }

判据：若在迭代过程中 |zn| > 2，则 c 一定不在集合中（逃逸半径为 2）。
在实际渲染中，记录逃逸所需的迭代次数并映射为颜色——迭代越多（越接近边界），颜色越"深"。
