文档隐写泄露体检

你发给客户的 Word 文件，可能比你以为的透明得多。
下载一份模拟文件，用 Word 亲眼看看里面藏着什么——或者上传你自己的 .docx，扫一遍再发出去。

元数据与机器名 修订记录 批注 白色文字层 文件不上传服务器

下载模拟文件

选一个场景，生成一份真实的 .docx 文件。用 Microsoft Word 或 WPS 打开， 在「审阅」菜单里看修订记录和批注，在「文件 → 属性」里看元数据。

下载后怎么看
  1. 用 Microsoft Word 打开文件
  2. 点顶部「审阅」选项卡 → 「显示标记」→ 勾选「修订」和「批注」
  3. 点「文件」→「信息」→「属性」查看元数据
  4. 全选文字（Ctrl+A），看看能复制出什么

扫描你的文件

纯前端，不上传

上传任意 .docx 文件，在你的浏览器里完成解析。 文件不会发送到任何服务器，离线也能用。

拖拽 .docx 文件到这里
或者点击选择文件

元数据

作者名、机器名、创建时间、最后保存者——这些字段默认全部保留在文件里，导出 PDF 时也会一起带走。

修订记录

接受修订后正文看起来干净，但历史改动还在文件里。接收方可以在「审阅」菜单里把所有改动翻出来。

批注

批注气泡在打印预览里可以隐藏，但文件里还在。批注里常常写着内部讨论口径，比正文更敏感。

白色文字

把字体改成白色，页面上看不见，但文本层还在。全选复制，或者换个背景色，内容就出来了。
