EXIF 是什么

EXIF（Exchangeable Image File Format）是嵌入在每张照片里的隐藏数据。你的手机每次拍照，都会自动写入：

GPS 坐标精确到门牌号
设备型号iPhone 15 Pro Max
拍摄时间精确到秒
镜头参数焦距、光圈、ISO
朝向拍照时面朝的方向
海拔你所在的楼层高度

这些数据对你不可见，但任何拿到原图的人都能用免费工具读取。一张在家拍的自拍，就能暴露你的家庭住址。

怎么保护自己

1. 关闭相机定位权限 设置 → 隐私 → 定位 → 相机 → 永不
2. 发图前清除元数据 用本工具一键清除
3. 不要发原图 微信「发原图」= 发你的坐标
4. 截图代替原图 截图不包含原始 EXIF
