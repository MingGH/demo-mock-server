帕隆多悖论

1996年，西班牙物理学家 Juan Parrondo 发现了一个不可思议的现象：
两个必输的游戏，交替着玩，反而能赢钱。

游戏规则
两个简单的抛硬币游戏，赢了 +1 元，输了 -1 元。两个都是必输的。

游戏 A

抛一枚稍微不公平的硬币。赢的概率 49.5%，输的概率 50.5%。

P(赢) = 0.495

简单粗暴，长期必输。

游戏 B

看你当前资金是不是 3 的倍数：

是 3 的倍数 → P(赢) = 9.5%
不是 → P(赢) = 74.5%

看起来不错？但稳态下你会频繁踩到 9.5% 的坑。

悖论的核心：A 单独玩必输，B 单独玩也必输。但如果交替玩 A 和 B（比如 AABB 循环），你反而能赢钱。这不是魔法，是数学。

结论
为什么两个输的加起来能赢？
关键在于 Game B 的「棘轮效应」。

Game B 单独玩时，你的资金会自然地频繁落在 3 的倍数上（稳态概率约 38%），这时你用的是那个只有 9.5% 胜率的烂硬币。

但当你穿插 Game A 时，A 的随机扰动会把你的资金「踢」离 3 的倍数。这样你在 Game B 中就更多地使用 74.5% 胜率的好硬币。

A 的作用不是赢钱，而是改变你在 B 中的状态分布。

这就像一个棘轮（ratchet）：A 是随机的扰动，B 是有方向性的齿轮。单独都没用，但组合起来就能把随机运动变成定向运动。

现实中的帕隆多悖论
生物进化
细菌在两种不利环境间切换，反而比固定在一种环境中存活率更高。随机相位变异就是自然界的帕隆多策略。
投资组合
两只长期亏损的股票，定期再平衡后组合收益可能为正。波动率泵效应（volatility pumping）本质上就是帕隆多悖论。
布朗棘轮
费曼提出的思想实验：分子的随机热运动，通过不对称的棘轮结构，可以被转化为定向运动。帕隆多悖论正是它的离散版本。
一句话总结：帕隆多悖论告诉我们，在非线性系统中，「坏 + 坏 = 好」是完全可能的。关键不在于每个策略本身的好坏，而在于它们之间的相互作用如何改变了系统的状态分布。

下次有人告诉你「两个烂选择里挑一个」的时候，也许你该想想：能不能交替着来？
