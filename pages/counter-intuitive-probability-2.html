为什么会这样？

这叫非传递性（Nontransitivity）。在日常生活中，"比较"通常是传递的：如果小明比小红高，小红比小刚高，那小明一定比小刚高。

但骰子比大小不是单一维度的比较，每颗骰子的面值分布不同，导致了循环克制关系。

A vs B: A的9能赢B所有面，A的4能赢B的1 → A赢的概率 = 5/9
B vs C: B的8能赢C的3,5,7中的两个... → B赢的概率 = 5/9
C vs A: C的7能赢A的2,4，C的5能赢A的2,4 → C赢的概率 = 5/9

沃伦·巴菲特曾用非传递骰子和比尔·盖茨玩游戏，让盖茨先选骰子。盖茨研究了一会儿后说："你先选。"

为什么"至少一个男孩"变成了1/3？

两个孩子的所有可能组合（等概率）：

(男,男) (男,女) (女,男) (女,女)

条件A "老大是男孩"：排除后两种 → 剩 (男,男)(男,女) → 双男概率 = 1/2

条件B "至少一个男孩"：只排除(女,女) → 剩 (男,男)(男,女)(女,男) → 双男概率 = 1/3

关键区别：条件B保留了更多的样本空间。"至少一个男孩"是比"老大是男孩"更弱的条件，所以双男孩的比例更低。

贝叶斯定理的威力

假设10000人接受检测：

真正患病：10000 × 1% = 100人
其中检测阳性（真阳性）：100 × 99% = 99人

没有患病：10000 × 99% = 9900人
其中检测阳性（假阳性）：9900 × 1% = 99人

总阳性 = 99 + 99 = 198人
阳性中真正患病 = 99 / 198 = 50%

关键洞察：当疾病很罕见时，健康人的基数远大于患病人数。即使假阳性率很低（1%），乘以巨大的健康人基数，假阳性的绝对数量也很大。

这就是为什么医生通常不建议对低风险人群做罕见病筛查。

为什么需要这么多？

这是经典的优惠券收集问题（Coupon Collector's Problem）。

关键在于：集齐最后几张的难度远大于前几张。

已有0张 → 下一张新的概率 = 5/5 = 100%，期望 1 包
已有1张 → 下一张新的概率 = 4/5 = 80%，期望 1.25 包
已有2张 → 下一张新的概率 = 3/5 = 60%，期望 1.67 包
已有3张 → 下一张新的概率 = 2/5 = 40%，期望 2.5 包
已有4张 → 下一张新的概率 = 1/5 = 20%，期望 5 包

总期望 = 5×(1/5 + 1/4 + 1/3 + 1/2 + 1/1) = 5 × 2.283 ≈ 11.42 包

一般公式：n种卡片的期望购买数 = n × H(n)，其中 H(n) = 1 + 1/2 + 1/3 + ... + 1/n 是调和级数。

如果是集齐108张水浒卡？期望要买约 569包。

睡美人问题
哲学家至今争论不休的概率难题
实验：周日抛一枚公平硬币。

为什么两个答案都"对"？

这取决于你怎么理解"概率"：

1/2派（Halfer）：
硬币是公平的，没有新信息改变这个事实。
"被叫醒"不是新证据，因为无论正反面都会被叫醒。
所以概率仍然是 1/2。

1/3派（Thirder）：
三种等可能的醒来情况：
① 正面 + 周一（概率1/2 × 1 = 1/2）
② 反面 + 周一（概率1/2 × 1/2 = 1/4）
③ 反面 + 周二（概率1/2 × 1/2 = 1/4）
但从"醒来次数"角度：正面1次，反面2次 → 1/3

模拟结果支持1/3派：如果你统计所有醒来时刻，正面只占1/3。

但1/2派认为这是在偷换概念——把"实验次数"和"醒来次数"混淆了。

这个问题至今没有公认的正确答案，它触及了概率论的哲学基础。

概率的世界没有"显然"

非传递骰子告诉我们：优势不可传递
男孩女孩悖论告诉我们：条件不同，概率不同

集齐问题告诉我们：最后一步永远最难
睡美人问题告诉我们："概率"本身的定义就有争议
