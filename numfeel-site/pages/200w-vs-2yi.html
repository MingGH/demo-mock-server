📐 数学分析：期望值 vs 效用

1. 期望值计算

选A期望值 = 1.0 × 200万 = 200万
选B期望值 = 0.99 × 2亿 + 0.01 × 0 = 1.98亿

从期望值看，B是A的99倍！理性人应该选B，对吧？

2. 但人不是期望值机器

经济学家早就发现，人对财富的感知是边际递减的：
• 从0到200万，幸福感提升巨大（还清房贷、改善生活）
• 从200万到2亿，幸福感提升有限（已经财务自由了，再多也就那样）

3. 效用函数

假设效用函数是对数形式 U(x) = ln(x)：

选A效用 = ln(200万) = ln(2,000,000) ≈ 14.51
选B效用 = 0.99 × ln(2亿) + 0.01 × ln(1) = 0.99 × 19.11 + 0 ≈ 18.92

用对数效用算，B的效用仍然更高。但如果你当前资产为0，那1%的"什么都没有"意味着效用负无穷。

4. 关键问题：你能承受那1%吗？

这不是数学问题，是你的人生只有一次的问题。
• 如果你已经有1000万身家，选B没问题，输了也不影响生活
• 如果你负债累累，200万能救命，那1%的风险你承受不起

🎯 什么情况下选A/选B？

✅ 应该选A的人

  • 背着房贷车贷，200万能解燃眉之急
  • 风险厌恶型，睡不着觉比多赚钱重要
  • 这是你唯一一次机会
  • 200万已经能显著改变你的生活
  • 你无法承受"什么都没有"的后果

✅ 应该选B的人

  • 已有足够资产，输了也不影响生活
  • 200万对你来说只是"还行"
  • 你有多次类似机会（可以分散风险）
  • 你追求期望值最大化
  • 你能坦然接受1%的失败

💡 结论：没有标准答案

这道题的精妙之处在于，两个选择都是理性的。

选A的人不是傻，是风险厌恶——200万的确定性比1.98亿的期望值更有价值。
选B的人不是贪，是期望值最大化——99%几乎等于100%，值得一搏。

真正的问题是：你是谁？你的处境是什么？你能承受什么？

经济学告诉我们：理性不是追求期望值最大化，而是追求效用最大化。
而效用，因人而异。
