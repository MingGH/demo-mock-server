通用公式：换选正确率 = (N − 1) / N（当排除到只剩一个可选时）

一般情形推导（K < N−2 时也成立）

初始 N 个选项，随机选 1 个，再可靠排除 K 个错误选项（K ≤ N−2）。

  • 不换正确率 = 1 / N
  • 换选时剩余可挑的候选数 = N − 1 − K
  • 换选正确率 = (N − 1) / (N × (N − 1 − K))
  • 当 K = N − 2（只剩一个可选），退化成 (N − 1) / N
通用计算器：输入 N 和 K，立刻看两种正确率
填入 N 和 K，点计算。
为什么？一句话解释 + 展开推导

一句话：你第一遍随机猜的 A 没有携带任何信息，它的正确概率始终是 1/4；排除 B、C 后，它们原本承担的 2/4 概率全部转移到唯一剩下的 D 上，所以 D = 3/4。

展开：用贝叶斯看。记"排除 BC"为证据 E。

  • P(E | 答案=A) = 1/3（从 {BC, BD, CD} 三种排除组合里恰好选到 BC）
  • P(E | 答案=D) = 1（B、C 都是错的，且你已选 A 不会被排除，必排除 BC）
  • P(E | 答案=B) = 0，P(E | 答案=C) = 0（不可能可靠地排除正确答案）
  • P(答案=A | E) = (1/3 × 1/4) / (1/12 + 1/4) = 1/4
  • P(答案=D | E) = (1 × 1/4) / (1/3) = 3/4

关键前提：排除必须是"可靠的"——你确实能判断 B、C 是错的。如果排除本身可能误伤正确答案，优势就消失，A 和 D 退回各 1/2。
