为什么会亏

算术期望 vs 几何期望

算术期望：E = 0.5 x 9 + 0.5 x 0.1 = 4.55
看着不错，每按一次「平均」赚 3.55 倍。但这个平均值骗了你。

财富是乘法累积的

按两次的所有情况：

9 x 9 = 81 (概率 25%)
9 x 0.1 = 0.9 (概率 25%)
0.1 x 9 = 0.9 (概率 25%)
0.1 x 0.1 = 0.01 (概率 25%)

几何期望 = (9 x 0.1)^0.5 = 0.9^0.5 = 0.949
小于 1。每按一次，你的财富中位数在缩水。

凯利公式怎么看

f* = (p x b - q) / b
p=0.5, b=8(净赔率), q=0.5
f* = (0.5 x 8 - 0.5) / 8 = 0.4375

凯利公式说：最优下注是总资产的 43.75%。
但这个按钮强制你压上 100%，远超最优比例。

按 n 次之后

按 n 次，期望赢 n/2 次输 n/2 次
最终 = 初始 x 9^(n/2) x 0.1^(n/2) = 初始 x 0.9^n

按 10 次：0.9^10 = 0.35（剩 35%）
按 50 次：0.9^50 = 0.005（剩 0.5%）
按 100 次：0.9^100 = 0.00003（几乎归零）
结论
这是一个经典的期望值陷阱。

算术期望为正，但几何期望为负。
少数人连续 x9 会暴富，但绝大多数人会破产。

平均值被极少数幸运儿拉高，而你大概率不是那个人。

「期望值为正」不等于「应该参与」
当赌注是你的全部时，中位数比平均数重要得多。
