🧮 数学推导：答案约为 N × 31.78%

设 N = 100万，我们要砍 N 次腿。

核心思路：考虑某一只特定的鸡，它保持完好的概率是多少？

关键洞察：一只鸡完好 = 它从未被选中

设在第 k 次砍腿前：

  • Hk = 完好的鸡数量（2条腿）
  • Ok = 独腿鸡数量（1条腿）
  • 有腿的鸡总数 = Hk + Ok

每次砍腿时，从有腿的鸡中随机选一只：

  • 选中完好鸡的概率 = H / (H + O)
  • 选中独腿鸡的概率 = O / (H + O)
E[Hk+1] = Hk - Hk/(Hk + Ok)

这个递推关系可以用微分方程近似求解。当 N → ∞ 时：

完好鸡的比例 → 0.31784... ≈ 31.78%

因此，完好的鸡的期望数量：

E[完好的鸡] = 1,000,000 × 0.3178 ≈ 317,844

🎯 有趣的发现

这个常数 0.31784... 非常接近 1/π ≈ 0.31831，但并不完全相等！

数学之美：这个问题没有简单的闭式解，但通过递推或微分方程可以精确计算。

直觉理解：一只鸡保持完好 = 它从未被选中。每次被选中的概率约为 1/(有腿鸡数)，但有腿鸡数会随着独腿鸡变成无腿鸡而减少，这使得分析变得复杂。

如果题目改成"从所有腿中随机选一条砍掉"，那答案就是 N/e² ≈ 13.53%。但原题是"从有腿的鸡中选一只"，所以答案是 31.78%。
