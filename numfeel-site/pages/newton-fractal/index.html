模块 3 · 放大边界：「国界线」的无限复杂

越靠近边界，命运对初始值越敏感。放大观察，边界永远不会变成光滑曲线。点击下方预设坐标，画布会动画缩放到目标位置。

提示：放大后可以继续在上方画布上点击/双击进一步探索；右键或双指捏合可缩小回到全景。

模块 5 · 数学原理

牛顿迭代公式（求方程 f(z)=0 的根）：

zn+1 = zn - f(zn) / f'(zn)

对于 f(z) = z³ - 1：

zn+1 = zn - (zn³ - 1) / (3·zn²) = (2·zn³ + 1) / (3·zn²)

三个根分别为：1, e2πi/3, e4πi/3（即 1, -1/2+√3/2·i, -1/2-√3/2·i，在复平面上构成正三角形的三个顶点）

染色规则：每个像素代表复平面上的一个起点 z₀，反复代入公式得到一个序列 z₀ → z₁ → z₂ → …，看它最终收敛到哪个根。
迭代次数越少（收敛越快）颜色越亮，越多（越接近边界）颜色越暗——边界区域因此自动呈现暗色分形纹路。

关键洞察

牛顿迭代本来是寻找答案的方法。把所有起点同时画出来后，我们看到的却是答案之间的战争。离答案很远的地方，结果反而稳定；在几个答案势均力敌的地方，命运对初始条件极端敏感。
