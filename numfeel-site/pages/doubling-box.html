薛定谔的钱箱：50%翻倍or消失

50%概率变双倍，50%概率消失。放还是不放？放多少？数学告诉你最优解

问题：有一个神奇的盒子：
把钱放进去，50% 概率变成 2倍（翻倍）
也有 50% 概率钱会 消失不见

从概率学和经济学的角度，你该怎么做才能利益最大化？
放，但只放一部分
期望值为正（每次赚25%），但全押必死。
凯利公式最优解：每次放入总资产的 25%

数学分析：为什么期望值会骗人

算术期望值（看起来很美）
E = 0.5 × (+100%) + 0.5 × (-100%) = 0%？
等等，如果赢了是"变成2倍"，即收益+100%，输了是"消失"，即收益-100%。
但实际上赢了拿回2倍本金，所以期望回报 = 0.5 × 2 + 0.5 × 0 = 1.0（不赚不亏？）
但如果"翻倍"是净赚一倍（放100拿回200）
E = 0.5 × 2x + 0.5 × 0 = x
放入 x 元，期望拿回 x 元。期望值 = 本金，这是公平游戏。
但如果"翻倍"理解为放100变200（净赚100），那期望值确实是0。

关键问题：即使期望值为正或持平，全押也是灾难。
几何期望值（真正决定长期命运）
G = (2)^0.5 × (0)^0.5 = 0
全押的几何增长率 = 0。只要输一次，一切归零。
这就是为什么全押必死——不是概率问题，是数学必然。

凯利公式：最优下注比例

f* = p - q/b
p = 赢的概率 = 0.5，q = 输的概率 = 0.5，b = 赔率 = 1（赢了净赚1倍）
f* = 0.5 - 0.5/1 = 0%（公平游戏，凯利建议不玩）
但如果赔率更好呢？比如赢了变3倍（净赚2倍）
f* = 0.5 - 0.5/2 = 25%
如果赢了拿回3倍（放100拿回300），凯利最优比例是25%。

为什么不能全押？

1. 期望值陷阱

全押的算术期望值看起来不错：每次期望收益 = 0.5 × 翻倍 + 0.5 × 0。
但算术期望值不等于你实际会得到的结果。你只活一次，不是平行宇宙的平均值。

2. 遍历性问题

掷硬币是非遍历的：1000个人各玩一次的平均值 ≠ 1个人玩1000次的结果。
全押玩1000次，只要输一次就归零。而你一定会输——1000次全赢的概率是 2^(-1000) ≈ 0。

3. 几何增长率才是真相

长期重复博弈中，决定你命运的是几何平均收益率，不是算术平均。
全押的几何增长率 = (2×0)^0.5 = 0。凯利比例最大化的正是几何增长率。

实际应用：投资中的仓位管理

这个箱子问题本质上就是仓位管理问题。华尔街的量化基金、职业德州扑克玩家，都在用凯利公式的变体。

实战建议：用半凯利（凯利值的一半）更安全。因为现实中你永远不知道精确的胜率和赔率，半凯利牺牲一点增长速度，换取大幅降低的破产风险。

核心原则：永远不要把所有鸡蛋放在一个篮子里。不是因为鸡蛋会碎，而是因为数学不允许。

结论

50%翻倍、50%消失的箱子，放钱进去是对的，但绝不能全押。

如果赢了拿回3倍，最优策略是每次放入总资产的 25%。
如果赢了只拿回2倍（净赚1倍），这是公平游戏，凯利建议 不玩。

利益最大化的关键不是"放不放"，而是"放多少"。
这就是凯利公式的智慧：在贪婪和恐惧之间，找到数学上的最优平衡点。
