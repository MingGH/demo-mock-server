误差扩散过程

观看算法逐像素处理：每个像素被强制变为黑或白后，误差按权重分配给右方和下方邻居。

速度 中

为什么只有黑白也能看到灰度

01

量化决策

每个像素只有一个选择：变黑(0)还是变白(255)。没有中间地带。如果原值是 180，离 255 更近，所以变白。

02

误差不能凭空消失

180 变成 255，误差是 -75（变亮了 75）。这个误差不能吞掉——它被按比例分配给右边和下面的邻居像素。邻居变暗了一点来"偿还"。

03

空间积分欺骗视觉

人眼的空间分辨率有限。当黑白点足够密时，大脑自动对一小片区域"取平均"，感知到连续灰度。黑点密 → 暗区，白点密 → 亮区。

暴力在哪

没有频域分析，没有傅里叶变换，没有任何"聪明"的数学工具。就是逐像素从左上到右下扫一遍，每个像素做一次加减法和一次比较。整张图只遍历一次，O(n) 时间，O(1) 额外空间。这是你能想到的最暴力的图像处理：一个 for 循环解决一切。

核心代码

for (let y = 0; y < height; y++) {
  for (let x = 0; x < width; x++) {
    let old = pixels[y][x];
    let neu = old > 128 ? 255 : 0;  // 暴力二值化
    pixels[y][x] = neu;
    let err = old - neu;             // 计算误差

    // 把误差甩给邻居
    pixels[y][x+1]   += err * 7/16;  // →  右
    pixels[y+1][x-1] += err * 3/16;  // ↙ 左下
    pixels[y+1][x]   += err * 5/16;  // ↓  下
    pixels[y+1][x+1] += err * 1/16;  // ↘ 右下
  }
}

12 行代码。没有任何依赖库，没有矩阵运算，没有 GPU。一个双重循环，四次加法，完成一切。

实际应用

报纸与激光打印

墨点要么有，要么没有。报纸上那些看起来有灰度的照片，全靠抖动算法实现。

初代 Macintosh

1984 年的 Mac 只有黑白屏幕。Atkinson 抖动（Bill Atkinson 发明）让系统界面看起来有灰度层次。

Game Boy 打印机

热敏纸只能打黑点。Game Boy Camera 拍的照片通过抖动在小票纸上"还原"灰度。

激光雕刻 / CNC

激光要么烧要么不烧。把照片雕刻在木头上，靠的就是抖动后的黑白点阵控制激光开关。
