发生了什么

网页通过 document.addEventListener('copy/cut/paste', ...) 监听剪贴板事件。 这三个事件不需要用户授权，页面加载后自动生效。 你复制的文本内容、时间戳、操作类型，全部可以被 JavaScript 捕获。

背后的代码

只需要 6 行 JavaScript：

// 监听复制事件
document.addEventListener('copy', function(e) {
  const text = window.getSelection().toString();
  // text 就是用户复制的全部内容
  // 可以发送到服务器、做正则匹配、提取敏感信息...
  console.log('用户复制了:', text);
});

各平台的防护措施

iOS 14+：App 读取剪贴板时顶部弹出横幅提示。
Android 12+：App 读取剪贴板时底部弹出 Toast 提示。
Android 13+：剪贴板内容 1 小时后自动清除。
浏览器：Clipboard API 读取需要用户主动授权（但 copy/paste 事件监听不需要）。
