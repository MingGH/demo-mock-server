一行公式看懂绞杀

手续费是乘法消耗，不是加法。每次剩 99.9%，N 次后剩 0.999N。

WN = W0 × (1 − c)N
  • c = 单次双边成本率（A股约 0.1%：佣金万1~万1.5双边 + 印花税千0.5卖出 + 过户费忽略）
  • N = 总交易次数 = 频率 × 时间
  • 当 N 足够大，(1−c)N → 0，本金指数衰减到零

为什么"一点点"会变成"很多"

加法消耗是线性的：每次固定亏 25 元，1000 次亏 2.5 万，慢得能看见。乘法消耗是指数的：每次按比例扣，前几十次几乎无感，后期加速塌方——等你意识到不对，桶已经快空了。
