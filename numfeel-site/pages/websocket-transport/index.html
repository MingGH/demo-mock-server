HTTP 头部开销

每次请求携带头部约 900 B。低频场景占比小，高频轮询时累积可观。

WebSocket 握手

初始 HTTP Upgrade 握手约 1100 B，之后每条消息帧头仅 ~8 B。

连接内存

WebSocket 长连接每条约 48 KB 内核缓冲区；HTTP 短连接约 4 KB。

SSE：中间选项

Server-Sent Events 只能服务端推客户端，但跑在 HTTP 语义里，更轻量。
