📐 从泰勒展开到欧拉公式

1 e^x 的泰勒展开
ex = 1 + x + x²/2! + x³/3! + x⁴/4! + ...
这是自然指数函数在 x=0 处的泰勒级数展开
2 将 x 替换为 ix
eix = 1 + ix + (ix)²/2! + (ix)³/3! + (ix)⁴/4! + ...
利用 i² = -1, i³ = -i, i⁴ = 1 的周期性
3 分离实部与虚部
eix = (1 - x²/2! + x⁴/4! - ...) + i(x - x³/3! + x⁵/5! - ...)
实部恰好是 cos(x) 的展开，虚部恰好是 sin(x) 的展开
4 得到欧拉公式
eix = cos(x) + i·sin(x)
当 x = π 时：eiπ = cos(π) + i·sin(π) = -1 + 0 = -1
5 欧拉恒等式
eiπ + 1 = 0
五个最重要的数学常数，通过最基本的运算，完美地联系在一起

📜 历史足迹

1714年
罗杰·科茨首次发现了类似的关系式，但未能完全理解其意义
1748年
莱昂哈德·欧拉在《无穷分析引论》中正式发表 eix = cos(x) + i·sin(x)
1988年
《数学智力》杂志读者投票，欧拉恒等式被评为"最美丽的数学公式"
2004年
《物理世界》杂志调查，欧拉恒等式再次被评为"史上最伟大的方程"
